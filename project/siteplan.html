Site Plan: Naija: Stories & Sights

Overview Design

Site Overview

Site Name

Naija: Stories & Sights A compact informational site about Nigeria's culture, places to visit, and practical travel tips.

Short Description / Why I Picked It

This site highlights Nigeria's cultural richness, heritage, and travel destinations. I chose Nigeria because it's my home, I want to share personal knowledge and local stories while demonstrating responsive web skills.

Site Purpose

Provide a clean, accessible, and mobile-first 3-page site that:

  • Introduces Nigeria with key facts and a hero image (Home)
  • Explores culture: food, music, festivals, and daily life (Culture)
  • Showcases travel spots and practical visiting tips (Travel & Places)

Each page will be responsive, accessible, and optimized for performance.

Target Audience Scenarios

  1. "Where should I visit in Nigeria if I have 3 days?"
  2. "What foods or festivals should I try and when do they happen?"

These scenarios drive content and page structure.

Design Specifications

Color Schema

Using a professional palette that reflects Nigeria's vibrant culture while maintaining readability:

  • Primary: #1e3c72 (deep blue) — header background, primary buttons, nav bar
  • Accent: #fdd835 (warm yellow) — call-to-action buttons, link hovers, focus outlines
  • Background: #f5f7fa (light neutral) — page background
  • Text: #333333 (dark gray) — body text

Usage: Headers and nav use gradient with primary; CTA and hover use accent; body and cards use neutrals.

Typography

Google Font: Open Sans

  • Open Sans 400 → body text
  • Open Sans 700 → headings and navigation

Font loaded with display=swap for optimal performance. Using only these two weights to keep page load fast.
